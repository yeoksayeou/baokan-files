Title: Dalian Machine Tool Plant Designs New Machine Tool: Completes 7 Hours of Work of Ordinary Machine Tool in 38 Minutes
Author:
Date: 1956-07-13
Page: 2

Xinhua News Agency, Lüda, July 12th. Design personnel at the Dalian Machine Tool Plant recently completed the design work for the UT-001 type combined machine tool.

The UT-001 type combined machine tool is a specialized machine tool used to process the headstock of the 1616 type lathe. Workers only need to place the workpiece on the workbench of this machine tool and press a button. The machine head will automatically extend twenty-four boring bars, simultaneously performing roughing and reaming operations on twelve holes in the workpiece. After processing to a certain extent, the worker presses the button again, and the workbench can invert the workpiece to perform two additional cutting operations. In this way, a workpiece that would take seven hours to process with a universal boring machine can be completed in just thirty-eight minutes with the combined machine tool. All components of this combined machine tool have interlocking safety devices. If the button is pressed incorrectly, the machine tool will not operate, so even ordinary technicians can operate it.

This machine tool was designed by the design personnel of the Dalian Machine Tool Plant with the help of Soviet experts, referring to Soviet technical data.

### 大连机床厂设计新机床  38分钟完成普通机床7小时的工作

1956-07-13
第2版()
专栏：

    大连机床厂设计新机床
    38分钟完成普通机床7小时的工作
    新华社旅大12日电　大连机床厂设计人员最近完成了UT零零一型组合机床的设计工作。
    UT零零一型组合机床是用来加工一六一六型车床上的主轴箱的专用机床。工人只需要把加工件放到这台机床的工作台上，按一下电钮，机床的机头就自动伸出二十四根镗杆，同时对工作物上的十二个孔进行拉荒、扩孔两道工序的工作。加工到一定程度，工人再按一下电钮，工作台又能把工作物倒转过来，进行另外两道工序的切削。这样，用万能镗床需要七小时才能加工完的工作物，组合机床只要三十八分钟就能完成。这个组合机床的所有部件都有连锁保安装置，错按了电钮，机床不会动作，因此连普通的技术工人也能操纵。
    这种机床是大连机床厂设计人员参考苏联的技术资料，在苏联专家的帮助下设计出来的。
