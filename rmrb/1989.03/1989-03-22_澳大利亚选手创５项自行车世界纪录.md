Title: Australian Athletes Set 5 World Records in Cycling
Author:
Date: 1989-03-22
Edition: 3

Xinhua News Agency, Beijing, March 20th.

Australian athletes set 5 world records in cycling. According to foreign media reports, Australian athletes set 5 world records in cycling on March 19th. Martin Vinnicombe, the silver medalist of the Seoul Olympics, set a new world record in the 1000-meter indoor cycling race with a time of 1 minute 2.84 seconds during the Commonwealth Bank series in Launceston, Australia. World champion Stephen Pate broke 3 world records, including the longest-standing professional cycling record—the 200-meter standing start race record. Pate's new record for this event is 10.459 seconds. In the 1000-meter time trial, Pate set another world record with a time of 1 minute 4.14 seconds. His third world record was set in the 500-meter standing start indoor race, with a time of 27.45 seconds. The fifth record was set by Nathan Page in the 3000-meter race. This result is 4 seconds faster than the 3 minutes 30.45 seconds set by Australian Dean Wood 3 years ago.

### 澳大利亚选手创５项自行车世界纪录

1989-03-22
第3版(国内新闻·体育)
专栏：

　　澳大利亚选手创５项自行车世界纪录
    新华社北京３月２０日电  据外电报道，澳大利亚运动员１９日创造了５项自行车赛世界纪录。汉城奥运会银牌获得者、澳大利亚的马丁·温尼科贝在英联邦银行系列赛上，以１分０２秒８４打破男子自行车室内比赛１０００米的世界纪录。这次比赛在澳大利亚的朗塞斯顿举行。世界冠军斯蒂芬·佩特一人打破３项世界纪录。其中包括一项年头最长久的职业自行车赛纪录——２００米行进出发赛世界纪录。佩特在这个项目上创下的新纪录是１０秒４５９。在１０００米计时赛中，佩特以１分０４秒１４创下又一项世界纪录。
    他的第３项世界纪录是在室内５００米行进出发赛上创造的，成绩为２７秒４５。
    第５项纪录是由内森·佩奇在３０００米比赛中创造的。这一成绩比澳大利亚的迪安·伍德３年前创下的３分３０秒４５的成绩快了４秒钟。
