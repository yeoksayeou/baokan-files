Title: High-Speed Lockstitch New Sewing Machine
Author:
Date: 1960-12-07
Page: 3
Shanghai New Products

    High-Speed Lockstitch New Sewing Machine
    The Shanghai Huigong Sewing Machine Factory has newly designed a high-speed lockstitch sewing machine with a speed of over 3,500 stitches per minute. It can sew very thick fabrics, and the machine structure is very simple. Each machine saves thirteen kilograms of metal materials.
    This new type of sewing machine is being prepared for mass production.

### 高速平针新型缝纫机

1960-12-07
第3版()
专栏：上海新产品

    高速平针新型缝纫机
    上海惠工缝纫机厂新设计了一种高速平针缝纫机，每分钟速度达到三千五百针以上，很厚的布件也能缝纫，而且机件结构非常简单，每一台节约了十三公斤金属材料。
    这种新型缝纫机正在准备成批生产。
