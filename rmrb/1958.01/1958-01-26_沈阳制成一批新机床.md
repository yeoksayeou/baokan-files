Title: Shenyang Manufactures a Batch of New Machine Tools
Author:
Date: 1958-01-26
Page: 3

    Shenyang Manufactures a Batch of New Machine Tools
    (Our report) The Shenyang First, Second, and Third Machine Tool Factories have recently manufactured a batch of precision machine tools.
    China's first C868 precision lead screw lathe has been successfully trial-produced at the Shenyang First Machine Tool Factory. It can process various machine tool lead screws from thirty centimeters to two thousand five hundred centimeters. The Shenyang Third Machine Tool Factory recently produced China's first four-axis automatic lathe. This machine tool can increase production efficiency by ten to twenty times compared to ordinary lathes. The Shenyang Second Machine Tool Factory recently successfully trial-produced one of China's largest vertical drilling machines. The maximum diameter of the drilled holes it can process is seventy-five millimeters. The entire machine tool weighs three thousand five hundred kilograms. Some people call it the "King of Vertical Drills."

### 沈阳制成一批新机床

1958-01-26
第3版()
专栏：

    沈阳制成一批新机床
    本报讯　沈阳第一、第二、第三机床厂，最近制成一批精密机床。
    我国第一台C868型精密丝杠车床已经在沈阳第一机床厂试制成功，它能加工三十厘米到二千五百厘米的各种机床丝杠。沈阳第三机床厂最近制成了我国第一台四轴自动车床。这台机床能比一般车床提高生产效率十倍到二十倍。沈阳第二机床厂最近试制成功了一种我国目前最大的立式钻床。它所加工的钻孔，最大的直径可达七十五毫米。整个机床重达三千五百公斤。有人把它称作“立钻之王”。
