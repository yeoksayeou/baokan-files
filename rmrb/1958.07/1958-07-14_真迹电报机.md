Title: True Image Telegraph Machine
Author:
Date: 1958-07-14
Page: 1

Effort to Scale the Heights of Science and Technology

    True Image Telegraph Machine
    China's first true image telegraph machine was recently successfully trial-produced at the Beijing Long-Distance Telecommunications Bureau.
    When using this telegraph machine to transmit telegrams, simply place the written draft of the telegram on the machine, and the recipient can directly receive the original true image from the receiving machine. This not only eliminates the complicated procedures of decoding, punching, typing, transmitting, and re-translating into Chinese characters, but also completely eliminates transmission errors. (Xinhua News Agency)

### 真迹电报机

1958-07-14
第1版()
专栏：努力攀上科学技术的高峰

    真迹电报机
    我国第一台真迹电报机最近在北京市长途电信局试制成功。
    使用这种电报机传送电报，只要把写好的电报底稿放在电报机上，对方就可从收报机上直接收到原稿的真迹。这样，不仅可以省掉译码、凿孔、打字、传出、再译成汉字等繁复的手续，还可以彻底消除发报差错。　　（新华社）
